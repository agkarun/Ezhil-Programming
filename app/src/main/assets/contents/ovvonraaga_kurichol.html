ஒவ்வொன்றாக குறிச்சொல்
அடுத்து, ’ஒவ்வொன்றாக’ என்கிற ஒரு குறிச்சொல்லைப் பார்க்கவிருக்கிறோம். ஆங்கிலத்தில் இதனை 'For Each Loop' என்று அழைப்பார்கள். உதாரணமாக, ஒன்று முதல் ஆறு வரை உள்ள எண்களை அச்சிடவேண்டும் என்றால், அதை இப்படிச் செய்யலாம்:

************************************* எண்கள் = [1, 2, 3, 4, 5, 6] @(எண்கள் இல் இவ்வெண்) ஒவ்வொன்றாக பதிப்பி இவ்வெண் முடி *************************************

இங்கே நாம் ‘எண்கள்’ என்ற பட்டியலில் ஆறு எண்களை வைத்துள்ளோம். பின்னர் அவற்றை ஒவ்வொன்றாக எடுத்து ‘இவ்வெண்’ என்ற பெயரில் சேமிக்கிறோம், பின் அதனை அச்சிடுகிறோம். ஒருவேளை, ஆறுக்குப் பதில் நூறு எண்கள் இருந்தால்? அத்துணை நீளமாக எழுதுவது சிரமம் ஆயிற்றே! உண்மைதான். அதற்காகவே ‘எழில்’ மொழியில் ‘ஆக’ என்ற குறிச்சொல் உள்ளது. இதனைப் பயன்படுத்தினால், ஒவ்வொன்றாக எழுதாமல், அதனை ஒரு கணக்காகச் செய்துவிடலாம், இப்படி:

************************************* @(எண் = 1, எண் <= 100, எண் = எண் + 1) ஆக பதிப்பி எண் முடி *************************************

மிக எளிய நிரல்தான். ஒவ்வொரு வரியாகப் படித்தால் தெளிவாகப் புரிந்துவிடும்: * முதலில் ‘எண்’ என்ற பெயரில் 1 என்ற எண்ணைச் சேமிக்கிறோம் * பிறகு, ‘எண் <= 100’ என்று நிபந்தனை விதிக்கிறோம் * நிறைவாக, எண்ணை ஒவ்வொன்றாக அதிகரிக்கிறோம் * ஆக, ’எண்’ என்பது 1, 2, 3, 4, 5, 6 ... என படிப்படியாக அதிகரிக்கும், திரையில் அச்சிடப்படும் * அந்த ’எண்’ 101ஆக மாறியதும், ‘எண் <= 100’ என்ற நிபந்தனை தவறாகிவிடுகிறது, ஆகவே, அதற்குமேல் அச்சிடுவதை நிறுத்திவிடுகிறோம்

பயிற்சி
இப்போது, உங்களுக்கு ஒரு பயிற்சி: * ஒன்று முதல் ஆயிரம் வரை உள்ள எண்களில் ஒற்றைப்படை எண்களைமட்டும் திரையில் அச்சிடவேண்டும் * அடுத்தபடியாக, ஆயிரத்திலிருந்து ஒன்றுவரை உள்ள எண்களில் இரட்டைப்படை எண்களைமட்டும் திரையில் அச்சிடவேண்டும் * இதற்கு நீங்கள் "%" என்ற குறியீட்டைப் பயன்படுத்தலாம், அது இரு எண்களை வகுத்து, அதன்பிறகு மீதமுள்ள எண்ணைமட்டும் தரும். விடை:

************************************* பதிப்பி "ஒற்றைப்படை எண்கள்" @(எண்=1, எண் <=1000, எண் = எண் + 1) ஆக மீதம் = எண் % 2 @(மீதம் == 1) ஆனால் # ஒற்றைப்படை எண் பதிப்பி எண் முடி முடி பதிப்பி "இரட்டைப்படை எண்கள்" @(எண்=1000, எண் >= 1, எண் = எண் - 1) ஆக மீதம் = எண் % 2 @(மீதம் == 0) ஆனால் #இரட்டைப்படை எண் பதிப்பி எண் முடி முடி *************************************

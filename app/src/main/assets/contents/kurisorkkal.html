குறிச்சொற்கள்
எழில் மொழியில் நிரல் எழுதுவதற்கு ஏராளமான குறிச் சொற்கள் உள்ளன. இவற்றை நாம் அடுத்தடுத்த அத்தியாயங்களில் விளக்கமாகப் பார்க்கவிருக்கிறோம். இப்போதைக்கு, எழில் மொழியில் உள்ள முக்கியமான குறிச் சொற்களின் பட்டியலையும், அதற்கு இணையான ஆங்கிலச் சொற்களையும் இங்கே பார்த்துவிடுவோம்:

* ஆக : for * நிறுத்து : break * தொடர் : continue * பின்கொடு : return * ஆனால் : if * இல்லைஆனால் : elseif * இல்லை : else * தேர்ந்தெடு : select * தேர்வு : case * வரை: while * செய் : do * முடியேனில் : until * ஏதேனில் : otherwise * பதிப்பி : print * நிரல்பாகம் : function * முடி : end
